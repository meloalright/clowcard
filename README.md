# clowcard

Clow card divination in your terminal.

## Install

### Homebrew

```
brew tap meloalright/monkey-tap
brew install clowcard
```

### Cargo

```
cargo install clowcard
```

### Binary

Download from [Releases](https://github.com/meloalright/clowcard/releases).

## Usage

```
clowcard            random draw
clowcard --en       random draw (english)
clowcard --zh       random draw (chinese)
clowcard --ja       random draw (japanese)
clowcard --json     output as JSON
clowcard -i <id>    by id (shows both positions)
clowcard -V         version
clowcard -h         help
```
